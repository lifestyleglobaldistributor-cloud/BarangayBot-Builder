# BarangayBot-Builder
that lets Local Government Units (LGUs) and barangay captains build their own multi-agent chatbot. 
Key features to include:
Visual drag-and-drop interface for 4 design patterns
Live connector marketplace (DSWD API, PhilHealth, Pag-IBIG, LTO)
Event-driven responses (like calamity fund release alerts)
Should demonstrate Stream Governance and real-time context
Should be buildable in "10 minutes"
